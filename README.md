## How to install

1. Clone the repository
2. In `src/class/bot.ts` change the `API_ KEY` to your own Mistral API key
3. Run the following command :

```bash
yarn install
```

## How to run

```bash
yarn start
```

## How to build

```bash
yarn build
```
